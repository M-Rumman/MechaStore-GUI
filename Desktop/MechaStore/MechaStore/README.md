# MechaStore
A DSOOP project to browse UI components, add to cart, and checkout. Basically it's kind of a local store to sell and purchase components locally just like OLX. It is limited to a department in a University.


# ComponentsPortal

A small beginner OOP project to browse UI components, add to cart, and checkout.

## How to run (local)
1. Open the project in C++ Builder or your preferred C++ IDE.
2. Compile and run the project from the IDE.

## Structure
- `src/` — code
- `data/` — JSON data (users, components, orders)
- `docs/` — project spec and notes
